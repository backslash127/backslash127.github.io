layout broken?

you may notice the layout of the grid looks broken, that's not really my fault.

currently, i'm using the built in masonry row layout in CSS but it isn't fully supported in most browsers yet for some reason?

it's been unsupported for literal years at this point and i'm convinced they've just forgotten about it. but on the off-chance they haven't, i'll be waiting patiently for my gallery to look like how it's supposed to.

if you're wondering why i don't just use JS or something to make the grid properly viewable by everyone -- i have tried on-and-off for months to get ANY masonry grid resource to work but none of them have, ever.

on another note, if you want to actually view the gallery properly, then open firefox (or install it if you haven't), type "about:config" into the URL bar, and then type in "masonry" into the search bar there. it should be the only option that shows up. toggle that to true, come back to this page, then reload.
