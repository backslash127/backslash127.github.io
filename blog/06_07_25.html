remember when i said i wouldn't remake the website again

i don't even have an excuse for this one.

i remade the website again, for like, the 6th time... maybe the 8th time? i wouldn't know i lost count.

why? fuck if i know LMAO. i always feel like i could improve on my skills with web development and cleaning up my website makes me feel better.

and!!! and!! and!!! i do actually notice improvement. i'm being more efficient with my code, i'm organizing files better, and i'm even using some Javascript here and there made from scratch! (not like the, the uhm the game development website)

i always like seeing how far i can go with my abilities, and i would say i've done pretty well with this. everything looks nicer and i've eliminated most of the weird hacky techniques i was using before to acomplish certain things.

can i guarantee that i won't rewrite the website again? no. don't be surprised when in a few months you end up reading another blogpost about how i cleaned up the website again.
